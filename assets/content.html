Llama has features similar to SilentTime and it also has the additional functionality of
allowing the user to set the location for the phone to be silenced. The application then uses
base transceiver stations to determine the set locations and silent the phone accordingly.
Phone Silencer is an application that specifically tackles the forgetfulness of turning your
phone back to ringer mode from silent mode. What the app perceives is user often forgot to
switch his/her phone back to ringer mode after switching on silent mode and therefore allows
user to specify the silent duration he/she wants before the phone automatically switches back
to ringer mode.
Shush! is an application built with similar functionalities with Phone silencer.
Even though all these applications provide a certain degree of convenience to users to silent
their phone ‘automatically’, it requires much of the user’s actions and attention to manually
define periods for the phone to be silent. We feel that if users were to be caught in urgent or
sudden events, he/she might not even have the time to customize the settings in order for the
application to work properly. We also thought that these applications would only realize its
full potential when the user actually spends his/her time to customize it.
